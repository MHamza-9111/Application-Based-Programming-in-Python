# Shapes Guess Game 🎨

A simple Python **Tkinter** game where the player has to guess the correct shape.
This project is part of my **AI Diploma** under **Sir Wahaj** at **Aptech Pakistan, Shahrah-e-Faisal**.

---

## 📝 Description

* The game randomly selects a shape from the following list:
  `circle`, `square`, `rectangle`, `oval`, `diamond`
* The player has **3 attempts** to guess the correct shape.
* If the player guesses wrong, the game will show remaining attempts.
* A **Play Again** button allows restarting the game anytime.

---

## 🎮 How to Play

1. Run the Python script `SGG.py`.
2. Enter your guess in the text box next to **Enter Shape**.
3. Click **Guess** to see if you’re correct.
4. After 3 wrong attempts, the correct shape is revealed.
5. Use **Play Again** to start a new round.

---

## ⚡ Features

* Simple GUI using **Tkinter**
* Limited attempts (3)
* Input validation (only accepts listed shapes)
* Real-time result updates
* Easy to restart the game
